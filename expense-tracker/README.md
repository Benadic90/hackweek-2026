# Expense Tracker

A personal finance tracking application to record, categorize, and analyze expenses.

## Features

- Add, edit, and delete expenses
- Categorize expenses (Food, Transport, Housing, etc.)
- Set a monthly budget and receive alerts when exceeded
- Visual insights using Chart.js doughnut chart
- LocalStorage integration for saving data
- Clean and professional UI

## Tech Stack

- HTML, CSS, JavaScript
- Chart.js (via CDN)
