# Habit Streak Tracker

Hi Reviewer! 👋

This is my submission for the Frontend **Habit Streak Tracker** challenge (100 points).

I built a highly functional and visually clean habit tracking tool that runs completely in the browser, without requiring any backend server.

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript, `localStorage`

### ✨ Features Implemented
- **Add & Delete Habits:** Users can easily add new goals and remove old ones.
- **Persistent Storage:** Everything is saved to the browser's native `localStorage`. If you refresh the page or close the browser, your habits and progress remain intact.
- **30-Day Contribution Graph:** Inspired by GitHub's contribution graph, each habit generates a 30-day grid representing the last 30 days up to "Today". Users can click the squares to toggle completion.
- **Smart Streak Calculation:** The JavaScript algorithm calculates your current streak by counting backwards from today. It's smart enough to know that if you haven't completed today's habit yet, it shouldn't break an ongoing streak from yesterday!

### 🚀 How to Run Locally
Since it uses purely client-side technologies, no installation is needed!
1. Clone this repository.
2. Navigate to the `habit-streak-tracker` folder.
3. Open `index.html` in your browser.
