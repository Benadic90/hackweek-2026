# Live Collaborative Whiteboard

Hi Reviewer! 👋

This is my submission for the Live Collaborative Whiteboard challenge. I built a real-time multi-user drawing app where anyone with the link can draw together on the same canvas simultaneously.

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript, Node.js, Express, Socket.IO

### ✨ Features
- **Real-time Drawing:** Multiple users can draw on the same canvas at the same time using Socket.IO WebSockets.
- **Drawing History:** When a new user joins, they instantly see everything that's already been drawn.
- **Color Picker & Brush Size:** Fully customizable drawing tools.
- **Eraser Mode:** Toggle the eraser to correct mistakes.
- **Undo Support:** Undo your last stroke, synced across all clients.
- **Clear All:** Wipe the entire board for everyone.
- **Live User Count:** See how many people are currently on the board.
- **Touch Support:** Works on phones and tablets too.

### 🚀 How to Run Locally
1. Clone this repo
2. `cd collaborative-whiteboard`
3. `npm install`
4. `npm start`
5. Open `http://localhost:3000` in two browser tabs and start drawing!
