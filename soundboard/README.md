# Soundboard Challenge

Hi Reviewer! 👋

Thanks for reviewing my Soundboard project! 

I went for a "digital DJ launchpad" aesthetic for this one. It uses public audio CDNs so the sounds work instantly without needing to download local files.

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript

### ✨ Features Implemented:
- **12 Unique Sounds:** Dynamically generated pads using JavaScript.
- **Volume Slider:** Controls the `.volume` property of all audio elements in real-time.
- **Stop All Button:** Instantly pauses all tracks and resets their playback time to 0.
