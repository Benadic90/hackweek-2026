# Kanban Board Challenge

Hi Reviewer! 👋

Thanks for checking out my submission for the Kanban Board challenge. 

I decided to build this using pure Vanilla code to demonstrate core DOM manipulation and state management skills without relying on heavy frameworks. 

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript

### ✨ Features Implemented:
- **Card Creation:** You can add new cards via the simple modal.
- **Movement:** Built using the native HTML5 Drag and Drop API.
- **Persistence:** State is saved to `localStorage`, so everything stays exactly where you left it after a refresh.
